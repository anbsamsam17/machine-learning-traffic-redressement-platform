affichera dans le tableau « Captures par module » une fois présentes.

## Vidéo

- **GIF court** (autoplay dans le README) : déposer `demo.gif` ici.
- **Vidéo MP4 complète** : préférer le glisser-déposer du `.mp4` directement dans
  l'éditeur de README sur github.com (lecteur inline natif) plutôt que de versionner
  le binaire dans le dépôt.

> Note : les fichiers de ce dossier sont volontairement **inclus** dans le dépôt
> (le `.gitignore` ignore les PNG à la racine et `prototypes/`, mais pas `docs/`).
